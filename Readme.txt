
=======================================================
基本操作

・SinglePlay
	移動	：WASD
	照準操作：マウスカーソル
	撃つ	：左クリック

・MultiPlay
	移動	：左スティック
	照準操作：右スティック
	撃つ	：LB/RB

=======================================================
デバッグキー
	タイトル画面に遷移します：F1
	全ステージアンロック	：タイトル画面でESCを一度押して、Qキーを3秒以上押す
